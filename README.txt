Schudwig — Landing Page
=======================

This is a static landing page scaffold for Schudwig: a gallery of small games, tools, and interactive experiments.

How to run locally
------------------

- Option 1: Double‑click `index.html` to open it in your browser.
- Option 2: Serve via a local web server (recommended for smooth relative paths):

  - Python 3
    - macOS/Linux: `python3 -m http.server 5173`
    - Windows (PowerShell): `py -3 -m http.server 5173`
    - Then open `http://localhost:5173` in your browser.

Project structure
-----------------

- `index.html` — Markup for the landing page
- `styles.css` — Dark theme with red/blue accents and layout styles
- `script.js` — Interactions: scroll progress, reveal on scroll, tilt, parallax, cursor
- `assets/favicon.svg` — Favicon

Customization
-------------

- Edit colors in `styles.css` under the `:root` variables.
- Replace placeholder tiles in the Showcase section with real project links.
- Add more sections as needed by following the existing structure.
